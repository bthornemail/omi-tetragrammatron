# WebGL viewer

The `viewer/` directory contains a browser visualization for the deterministic digital twin.

## Run

Start the runtime HTTP server:

```bash
./opencode.bin --serve 8080
```

Open:

```text
http://127.0.0.1:8080/
```

## Files

```text
viewer/index.html          page shell, import map, controls hint
viewer/ring-loader.js      fetches /frame and /ring and polls for changes
viewer/geometry-decoder.js pure JavaScript geometry decoder mirroring C formulas
viewer/twin-viewer.js      Three.js scene, Polybius grid, Fano plane, Hopf vectors, spiral ring path
```

## HTTP endpoints

- `/` serves the viewer.
- `/frame` serves the JSON digital twin frame generated by the runtime.
- `/ring` serves ring state as JSON.

## Viewer role

The viewer is read-only visualization. It does not replace the deterministic C runtime. It consumes the runtime state, renders geometry, and uses browser-side animation as presentation rather than computation authority.
