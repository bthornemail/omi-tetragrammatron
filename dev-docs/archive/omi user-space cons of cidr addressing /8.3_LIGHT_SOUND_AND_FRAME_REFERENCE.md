# Light, Sound, and Frame Reference

Declarations are like points on the surface of a sphere. Each one says:

```text
this is asserted
this is intended
this is being placed into the world
```

But declarations alone can scatter.

Definitions provide boundaries:

```text
this is the meaning range
this is the frame
this is the condition of resolution
this is the closure rule
```

When declarations are bounded by definitions, they can resolve toward a centroid.

```text
declarations + definitions -> centroid of agreement
```

## Light and Sound

Light and sound are the two most ubiquitous human reference spectrums for describing this process.

```text
Omi-Light = projection spectrum
Omi-Sound = intention spectrum
```

Light shows how the declaration appears.
Sound carries how the declaration is intended.
The frame compares them.
The centroid resolves them.

```text
Light orients projection.
Sound centers intention.
Definitions bound declaration.
The Omi-Ring resolves the centroid.
```

## Frame Reference

The frame does not ask which projection is loudest or brightest. It asks whether the declaration can be bounded, validated, resolved, recorded, and replayed.

```text
projected signal
  -> bounded frame
  -> declared intention
  -> definitional comparison
  -> receipt-backed centroid
```
